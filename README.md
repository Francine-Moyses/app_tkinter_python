## Projeto | Aplicativo em python com interface gráfica Tkinter
**Tema:** Aplicativo com interface gráfica tkinter para tratamento de dados com python

**Descrição:**
* Função: Tratamento, estruturação e visualização de dados com linguagem de programação python
* Recursos: Atualização e exportação de dados, visualização de arquivos em tabela dinâmica, filtro por entrada de texto e caixa de combinação.
